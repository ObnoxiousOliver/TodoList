# TodoList

**-- INSTALL TODO LIST --**

1. Download ["TodoListInstaller.zip"](TodoListInstaller.zip)
2. Extract zip
3. Open "TodoListInstaller.exe"
4. Choose directory / version
5. Hit INSTALL

**-- UPDATE TODO LIST --**

1. Download ["TodoListInstaller.zip"](TodoListInstaller.zip)
2. Extract zip
3. Open "TodoListInstaller.exe"
4. Check OVERRIDE CONTENT
5. Choose directory of already installed version
6. Choose version
7. Hit INSTALL
